Program Overview

The New Media Design and Web Development Diploma is an 18-month full time-program that equips students with creative and technical skills in various new media areas. In this program, students enage in hands-on learning with cutting-edge technology, courses cover:

Web Design and Development

This course will introduce students code management, and familiarize them with the tools of web design. Students will be assessed by a combination of quizzes, and individual projects.

Graphic Design

Students will learn the theory and practice of design and layout to visualize and develop problem-solvig ideas and render effective content using the principles of design, typograhy, and colour in layouts for print and web products.

Information Architecture

This course is an extensive discipline involving the organization, labeling, navigation, and searching systms for web and interactive products that enable users to find and manage information more successfully.

New Media communications and storytelling

Student will explore how people read scree text in contrast to printed text, view examples of high and low-impact content, and practice adapting traditional print media to new media publications.
